## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 800x600 pixels (4:3 aspect ratio)
- **Max File Size**: 500KB per image (for fast loading)
- **Quality**: High quality, clear images

## How to Add Images

1. Place your image files in this folder (`public/images/social-work/`)
2. Name them according to the convention above
3. The images will automatically appear on the website
4. If an image is missing, that activity will display without an image

## Image Optimization Tips

Before adding images:
- Compress images using tools like TinyPNG or ImageOptim
- Use JPG for photos, PNG for graphics with transparency
- Consider using WebP format for better compression
- Ensure images are properly cropped and centered

## Testing

After adding images:
1. Run `npm run dev` to start the development server
2. Navigate to the Social Work section
3. Verify all images load correctly
4. Check on mobile devices for responsive behavior
